# AI Reels Factory — инструкция

Программа делает рекламные Reels из текста: вставляешь сценарий → получаешь готовое видео
с озвучкой, титрами и музыкой. Можно работать через окно программы или через своего Telegram-бота.

---

## 1. Установка

### Mac
1. Скачай **AI-Reels-Factory-…-arm64.dmg** со страницы релизов.
2. Открой файл → перетащи «AI Reels Factory» в папку «Программы».
3. Первый запуск: **правый клик по программе → «Открыть» → «Открыть»** (так один раз, потому что
   программа пока без подписи Apple; дальше открывается обычным двойным кликом).

### Windows
1. Скачай **AI-Reels-Factory-Setup-…exe**.
2. Запусти. Если выскочит синее окно «Windows защитила ваш компьютер» →
   **«Подробнее» → «Выполнить в любом случае»** (это разовое предупреждение для новых программ).
3. Установится и откроется.

---

## 2. Первый запуск — настройка (5 минут, один раз)

Программа сама проведёт по шагам:

| Шаг | Что нужно | Где взять |
|-----|-----------|-----------|
| 1. Telegram-бот | токен бота | напиши [@BotFather](https://t.me/BotFather) → `/newbot` → придумай имя → он даст токен. Нажми «Проверить» → должна загореться ✓ |
| 2. Озвучка | ключ ElevenLabs | [elevenlabs.io](https://elevenlabs.io/app/settings/api-keys) → раздел API Keys → создать ключ |
| 3. Кадры | ключ Pexels (бесплатно) | [pexels.com/api](https://www.pexels.com/api/) → Get API Key |

После «Готово» — можно делать видео.

> Без шага 3 (Pexels) и без своих клипов из Google Drive видео не из чего собрать —
> добавь хотя бы Pexels.

---

## 3. Как сделать видео

**Вариант А — через окно:** вкладка «Создать видео» → вставь сценарий → «Создать видео».
Через 1–2 минуты ролик появится во вкладке «Мои видео».

**Вариант Б — через Telegram:** просто напиши сценарий своему боту. Через пару минут он пришлёт
готовый ролик прямо в чат.

⚠️ **Важно:** программа должна быть **запущена** (она живёт в фоне, значок в трее/панели).
Если компьютер выключен или программа закрыта — бот не ответит. Включил компьютер → бот снова работает.

---

## 4. Свои клипы из Google Drive (по желанию — чтобы видео были «твои»)

1. Сложи свои видео-клипы (и скринкасты продукта) в одну папку в Google Drive.
2. Сделай папку **«Доступ по ссылке»** (в Drive: правый клик по папке → «Открыть доступ» →
   «Все, у кого есть ссылка»).
3. В программе: «Настройка» → вставь ссылку на папку + Google API key
   ([как создать](https://console.cloud.google.com/apis/credentials) — включить «Google Drive API»).
4. Нажми **«Собрать каталог»** — программа сама скачает клипы и разложит их по смыслу.
   Если добавишь ещё и ключ OpenAI — клипы будут подбираться умнее (по содержимому кадра).

---

## 5. ИИ-генерация кадров (по желанию — уникальные планы)

«Настройка» → раздел «ИИ-генерация»: включи **FAL.ai**, вставь FAL-ключ
([fal.ai/dashboard/keys](https://fal.ai/dashboard/keys)), выбери модель (Kling / Veo / Runway)
и лимит клипов на ролик. Тогда уникальные кадры, которых нет в стоке, будут генерироваться ИИ.
Это платно за клип, поэтому есть лимит.

---

## 6. Если что-то не так

| Симптом | Что сделать |
|---------|-------------|
| Бот не отвечает | проверь, что программа запущена и компьютер включён; в «Настройке» нажми «Проверить» у токена |
| «Нет источников кадров» | добавь ключ Pexels или папку Google Drive в «Настройке» |
| Видео долго делается | это норма: рендер 1–2 минуты (длинный сценарий — дольше) |
| Mac: «программу нельзя открыть» | правый клик по программе → «Открыть» |
| Появилась плашка «обновление» | нажми «Перезапустить и обновить» — это новая версия |

---

Обновления приходят сами — когда выйдет новая версия, программа предложит обновиться одной кнопкой.
